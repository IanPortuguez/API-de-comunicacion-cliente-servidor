This API, built with **Django Rest Framework**, **converts JSON files into Python objects**, stores them as relational database models, and provides endpoints for efficient data querying.

## Project Features

- **Data Transcription**: Converts JSON files into Python objects for efficient processing and manipulation.

- **Base64 Handler**: Converts Base64-encoded images into local files without storing them in the database.

- **Relational Database Creation**: Automatically generates relational database models from the structure of incoming data, optimizing organization and storage.

- **Data Querying**: Provides a `/buscar` endpoint to query information that has been previously processed and stored in relational models.

- **Client-Server Communication**: Receives data from external applications and websites, processes it, and integrates it into a relational database model.
